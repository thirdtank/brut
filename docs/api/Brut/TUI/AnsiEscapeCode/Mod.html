Overview

Defines methods for each known code. This module can be included into other classes so you can write self.ansi.bright_blue (e.g.) Note that Brut::TUI::AnsiEscapeCode extends this module so you can always do Brut::TUI::AnsiEscapeCode.ansi.bright_blue.

#ansi(*name) ⇒ Object|Brut::TUI::AnsiEscapeCode

Method for accessing the pre-defined ANSI escape codes. This method works in two ways: RGB mode and predefined mode based on the arguments passed.

Examples:

Using predefined codes

Brut::TUI::AnsiEscapeCode.ansi.red
Brut::TUI::AnsiEscapeCode.ansi.bold
Brut::TUI::AnsiEscapeCode.ansi.underline

Using RGB codes

Brut::TUI::AnsiEscapeCode.ansi(87, 255, 128)

Using code names

Brut::TUI::AnsiEscapeCode.ansi(:bright_blue)

Parameters:

  • name (String|Symbol|Array<Integer>) —

    If called with no arguments, return #object, allowing you to call a dynamically-defined method based on the AnsiEscapeCode names in CODES. If called with a String or Symbol, will return the AnsiEscapeCode for that name from CODES. Otherwise, this should be exactly three integers, each between 0 and 255 that represent red, green, and blue, respectively. In this case, the ANSI escape code for an RGB value is returned.

Returns:

  • (Object|Brut::TUI::AnsiEscapeCode) —

    The corresponding ANSI escape code object or the special #object.

def ansi(*name)
  case name
  in [ r, g, b ]
    Brut::TUI::AnsiEscapeCode.new("rgb(#{r},#{g},#{b})", "38;2;#{r};#{g};#{b}")
  in []
    object
  else
    CODES.fetch(name[0].to_sym)
  end
end

def object
  @object ||= begin
                object = Object.new
                CODES.each do |name, code|
                  object.define_singleton_method(name) do
                    code
                  end
                end
                object
              end
