A Utility-Based CSS Framework For Getting Started

⚠️

This documentation is in somewhat rough shape. What information is here is accurate, however it is not as complete or elucidating as it could be.

The .css, however, is much more stable and should be considered as a release candidate, unlikely to be given breaking changes in the near future.

BrutCSS lets you get started quickly designing in the browser. It does this with classes whose names are based on the CSS attributes and values they will apply. You can quickly learn BrutCSS to start designing in minutes.

BrutCSS does not replace the use of CSS. BrutCSS has no build step, no derived classes, and does not attempt to provide a class for every CSS attribute/value combination.

BrutCSS knows you know CSS, knows you know your web framework, and wants you to use it.
